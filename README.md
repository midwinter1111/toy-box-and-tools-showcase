# toy-box-and-tools-showcase

Vue 3 をベースとした実装のお試しで作成したおもちゃやツール群をまとめるリポジトリです。
Vite, Tailwind CSS v4, Storybook, Vitest などの技術スタックを利用しています。

## 🏗️ 技術スタック（暫定、追加・変更予定あり）

Framework: Vue 3.5+ (Script Setup)

Build Tool: Vite

Store: Pinia

Routing: Vue Router

HTTP Client: Axios

Mock: MSW (Mock Service Worker)

Styling: Tailwind CSS v4

Testing: Vitest (UI, Browser Mode, Coverage)

Design/Dev: Storybook, Vue DevTools

Lint/Format: ESLint, Oxlint, Oxfmt
